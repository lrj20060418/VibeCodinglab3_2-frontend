# Frontend（Vue 3 + Vite）

## 安装与运行

```bash
cd frontend
npm install
npm run dev
```

默认开发地址通常是 `http://localhost:5173/`。

## 常见问题

如果安装依赖时出现 `ENOSPC: no space left on device`，请先清理磁盘空间后重试（例如清理临时文件、删除不必要的下载/缓存等）。
